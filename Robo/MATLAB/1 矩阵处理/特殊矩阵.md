# 通用性特殊矩阵
这几种矩阵函数调用格式相似：
```matlab
zeros(m); %  m x m 零矩阵
zeros(m,n); %  m x n 零矩阵
zeros(size(A)); %  与 A 同样大小零矩阵
ones(); % 用法跟 zeros 差不多

%%
eye();

%%
rand(); % (0,1) 开区间均匀分布的随机数构成的矩阵，参数形式跟上面差不多
		% fix(a + (b-a+1)*x) [a,b] 区间上均匀分布的随机数
randn(); % 产生均值为 0，方差为 1 的标准正态分布随机数 x
		% \mu + \sigma^{2}x
```
