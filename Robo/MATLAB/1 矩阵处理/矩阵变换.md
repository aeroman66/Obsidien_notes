对一个矩阵进行某种变换与操作：
# 对角阵
对角矩阵：只有对角线上有非零元素
数量矩阵：对角线上元素相等的对角矩阵
单位矩阵：对角线上元素 = 1 的对角矩阵
- 提取矩阵对角线元素
```matlab
diag(A) % 提取矩阵 A 主对角线元素，产生一个列向量
diag(A,k) % 提取矩阵 A 第 k 条对角线元素，产生一个列向量
		% 对角线约定，主对角线往上分别为 1 2 3...条,
		% 往下分别为 -1 -2 -3 条
```
- 构造对角阵
```matlab
diag(v) % 以向量 V 为对角线元素，产生对角矩阵
diag(V,k) % 以向量 V 为第 k 条对角线元素，产生一个对角矩阵
```
# 三角阵
上三角阵、下三角阵
```matlab
triu(A) % 提取矩阵 A 的主对角线及以上的元素
triu(A,k) % 提取矩阵 A 的第 k 条对角线及以上的元素

tril() % 调用格式与上面完全相同
```
# 转置
```matlab
A.' % 转置
A' % 复共轭转置
```
# 旋转
```matlab
rot90(A,k) % 将矩阵 A 沿逆时针方向旋转 90 度的 k 倍
```
# 翻转
```matlab
fliplr(A) % 对矩阵 A 左右进行翻转
flipud(A) % 对矩阵 A 上下进行翻转
```
# 求逆
```matlab
inv(A) % 求 A 矩阵的逆
```
